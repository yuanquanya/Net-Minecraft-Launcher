04 个核心决策
C++
01 ──────
C++ 原生后端 · 本地 HTTP 服务

NMCL 的核心由 C++ 编写，启动后在本地运行一个轻量 HTTP 服务器。前端是标准 HTML/CSS/JS，通过 HTTP 请求与后端通信。没有 Electron，没有打包 Chromium，安装包极小，内存占用极低。架构透明，逻辑清晰。

C++ CORE LOCAL HTTP NO ELECTRON
02 ──────
HTML 前端
随时可改

界面层就是标准 web 文件，想改？直接编辑，刷新即生效。对开发者和贡献者极其友好。

WEB UI
03 ──────
Java 环境
自动部署

缺 Java？版本不对？NMCL 会自动下载并配置合适的 JDK 环境，一键解决运行库烦恼。

AUTO-DOWNLOAD
